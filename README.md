# <p align="center"> VolaOS
### <p align="center"> Простая open-source операционная система, написанная на C.

## Цели
На данный момент ОС, по факту, является пустым ядром, которое умеет выводить надпись на экран, и ничего более. Главная задача сейчас: доделать ISR. Сама таблица дескрипторов прерываний была добавлена, но в ней пока что только одно прерывание (Divide by 0).

## Запуск
Предполагается, что у вас уже установлена QEMU. Инструкция для VirtualBox будет позже. В релизах и так идёт .iso файл, который можно установить хоть на сам ПК.<br>
Скачайте .iso файл из релиза и выполните указанные ниже команды:
```sh
qemu-system-i386 -cdrom vola.iso
```
